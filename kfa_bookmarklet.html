joinkfa.com 로그인 후 선수 목록에서 데이터를 자동 추출합니다

🔖 북마클릿 설치 방법

1

아래 「KFA 선수 추출」 버튼을 북마크바로 드래그하거나
우클릭 → 즐겨찾기에 추가 하세요

아래 버튼을 브라우저 북마크바로 드래그하세요

⚽ KFA 선수 추출
💡 북마크바가 안 보이면: Cmd+Shift+B (Mac) / Ctrl+Shift+B (Windows) 로 표시

🌐 KFA 사이트 이동 방법

2

joinkfa.com에 접속 후 로그인하세요

3

리그/대회 선택 → 2026 K6 대전광역시 디비전 리그 [k6 대전광역시] 클릭

4

참가팀 탭 → 대신FC 팀 클릭 → 선수 탭 클릭

5

선수 목록이 화면에 표시된 상태에서 북마클릿 버튼을 클릭하면
자동으로 선수 데이터가 JSON으로 추출됩니다

📋 데이터 가져오기

6

추출된 JSON 팝업에서 「클립보드에 복사」 버튼 클릭

7

스타팅 XI 메이커로 돌아와서
좌측 사이드바 → 「📥 JSON」 버튼 → 붙여넣기 → 가져오기

⚠️ KFA 사이트 구조에 따라 자동 추출이 안 될 수 있습니다.
그럴 경우 선수 추가 탭에서 직접 입력하거나,
아래 JSON 형식으로 직접 작성하여 가져오기 하세요.

✍️ 수동 입력 JSON 형식

선수 목록을 직접 아래 형식으로 작성 후 앱에 가져오기 할 수 있습니다: 

[
  {"no": 1,  "name": "홍길동", "pos": "GK"},
  {"no": 2,  "name": "김철수", "pos": "DF"},
  {"no": 5,  "name": "이영희", "pos": "DF"},
  {"no": 8,  "name": "박민준", "pos": "MF"},
  {"no": 10, "name": "최성현", "pos": "FW"},
  {"no": 11, "name": "정우성", "pos": "FW"}
]

포지션 코드:
  GK = 골키퍼
  DF = 수비수
  MF = 미드필더
  FW = 공격수
← 스타팅 XI 메이커로 돌아가기
